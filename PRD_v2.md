# 🎨 趣味英语语法绘本 —— 交互式学习网页

## 一、项目概述

基于《趣味英语语法》第1-4册的全部内容，制作一个**精美的交互式英语语法学习网页**。
不是答题/测验形式，而是一本可以"翻阅"的**数字语法绘本**。
面向小学三年级学生，通过微信链接即可打开。

## 二、设计理念

不要考试，不要测验，不要做题。
让语法学习像**翻看一本精美的绘本**一样自然有趣。

**核心交互模式：**
- 📖 **书式浏览**：左右滑动/点击翻页，每页讲一个语法概念
- ✨ **即点即学**：点击句子中的高亮词，弹出解释气泡
- 🎨 **视觉化语法**：每个语法规则配色彩标注+小动画
- 🐱 **角色引导**：一只叫"Grammy"的小猫（🐱）全程陪伴讲解
- 🎯 **渐进探索**：从简单到复杂，4册书的内容按章节组织

## 三、参考设计风格

**参考对象：**
1. **Duolingo** — 色彩鲜明、角色驱动、进度可视化
2. **Khan Academy Kids** — 干净、温暖、动物角色

**设计关键词：**

## 六、技术规格

### 前端技术
- **单页 HTML 文件**（所有 CSS/JS 内联）
- 纯前端，无后端，无数据库
- 响应式设计：手机优先（320px+），适配平板（768px+）
- 兼容微信内置浏览器

### 性能要求
- 首屏加载 < 2s（Google Fonts 外）
- 所有图片用 CSS/emoji/Unicode 代替
- 总大小 < 500KB

### 数据持久化
- localStorage 存储：
  - 已完成的课程列表
  - 每课获得的星星数
  - 当前阅读位置（第几册第几页）
  - 最后一次学习时间

### 交互库
- **不使用任何外部 JavaScript 库或框架**
- 纯原生 JS (ES6+)
- CSS 动画使用 @keyframes + transition
- 翻页效果用 CSS transform

## 七、设计细节要求

### 视觉层级
```
第1层：标题/大数字 → 字体粗细900，颜色 #FF6B6B
第2层：课程名称 → 字体粗细700，颜色 #2D3436
第3层：正文/解释 → 字体粗细400，颜色 #2D3436
第4层：辅助信息 → 字体粗细400，颜色 #636E72
第5层：标注/标签 → 字体粗细600，颜色 #4ECDC4
```

### 动画效果
- 翻页：平滑滑入（0.3s ease）
- 点击弹出解释：从下往上飞入（0.25s spring）
- 星星收集：放大+旋转（0.5s bounce）
- 拖拽填空：弹性回弹（0.3s elastic）
- Grammy 猫咪：轻微上下浮动（2s infinite）

### 字体
```css
/* 标题: bold 圆润 */
font-family: 'Noto Sans SC', sans-serif;
/* 英文示例: 清晰易读 */
font-family: 'Courier New', monospace; /* 代码风格英文 */
```

### 响应式断点
```css
/* 手机 (< 480px) */  /* 默认 */
/* 大手机 (480-768px) */  @media (min-width: 480px)
/* 平板 (768px+) */  @media (min-width: 768px)
```

## 八、特殊功能

### 1. Grammy 猫咪助手 🐱
- 每课开头出现，用气泡说话
- 讲解语法点时出现
- 答对互动时跳舞庆祝
- CSS 绘制的猫咪（不用图片）

### 2. "语法放大镜" 🔍
- 句子中可点击的词高亮显示
- 点击后弹出解释卡片
- 解释卡片包含：词性、中文意思、例句

### 3. "魔法变换器" 🔄 
- 滑块/按钮控制句子变换
- 如：单数↔复数、肯定↔否定、现在↔过去
- 拖动滑块看到句子实时变化

### 4. "单词银行" 🏦
- 学过的语法词自动收集
- 可在"单词银行"页面查看
- 显示已掌握的词数

## 九、非功能需求

1. 所有交互都适合儿童手指（点击区域 ≥ 48px）
2. 字体大小 ≥ 16px，防止 iOS 缩放
3. 触摸操作流畅，无卡顿
4. 支持深色模式（可选）
5. 离线可用（Service Worker，可选）
